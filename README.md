# Supermart SQL Data Analysis Project

## Project Overview
This project focuses on analyzing a Supermart sales database using SQL. 
The goal is to extract meaningful business insights related to sales performance, customer behavior, product performance, and profitability.

## Tools & Technologies
- PostgreSQL
- pgAdmin
- SQL (Joins, Aggregations, Window Functions)

## Key Analysis Performed
- Sales performance by region
- Top selling products and categories
- Discount impact on profit
- Customer segmentation analysis
- Monthly and cumulative sales trends
- Ranking products and customers using window functions
- Moving averages and running totals
- Profit analysis by customer and region

## SQL Concepts Used
- INNER JOIN
- GROUP BY & HAVING
- ORDER BY
- Window Functions:
  - RANK()
  - DENSE_RANK()
  - ROW_NUMBER()
  - LAG()
  - AVG() OVER()
  - SUM() OVER()

## Key Insights
- Identified top performing regions and products
- Analyzed customer contribution to total sales
- Evaluated discount impact on profitability
- Ranked customers and products based on sales and profit

## How to Run
1. Import the Supermart database into pgAdmin
2. Execute SQL scripts from the `SQL_Scripts` folder
3. Review query outputs and insights
